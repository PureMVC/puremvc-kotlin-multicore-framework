# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] 2025-02-25

### Changed

- Migrated build system to Kotlin DSL
- Migrated unit tests to Kotlin (kotlin.test)
- Migrated documentation to Kotlin-based Dokka system

## [1.3.0] - 2023-02-08

### Removed

- Removed JUnit Dependency from the implementation

## [1.2.0] - 2023-01-21

### Changed

- Refactoring Mediator to accept start projected WeakReferences

## [1.1.0] - 2020-04-11

### Changed

- Refactoring Mediator to accept WeakReferences

## [1.0.0] - 2020-04-11

### Added

- Brand new implementation of ported code, equivalent to AS3 MultiCore Version 1.0.5.
